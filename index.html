Recycling Sorter

This AI tool detects if an item is Plastic, Paper, or Not Recyclable using live camera input.

Who Benefits?

Students, teachers, and anyone learning about recycling sorting and machine learning.

What It Does?

Uses a Teachable Machine model to classify items into 3 categories in real time.

Instructions

Press Start, show an object to the camera, and watch the prediction cards highlight with confidence bars.

Why It Matters

Correct recycling helps reduce landfill waste, saves energy, and protects the environment. AI tools like this can help people learn how to sort waste properly.
